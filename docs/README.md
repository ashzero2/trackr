# Privacy Policy Page (GitHub Pages)

This folder contains a static privacy policy page for Play Console submission.

## Files

- `index.html` - Privacy policy page

## Host on GitHub Pages

1. Push this branch to GitHub.
2. Open repository settings on GitHub.
3. Go to **Pages**.
4. Under **Build and deployment**, choose:
   - **Source**: Deploy from a branch
   - **Branch**: `main`
   - **Folder**: `/docs`
5. Save and wait for deployment.

Your privacy policy URL will look like:

`https://<your-github-username>.github.io/<repo-name>/`

Use that URL in Google Play Console.
